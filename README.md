# Python for geosciences

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/rguilcas/python-geosciences/main?labpath=Welcome.ipynb)

Course on python libraries for geosciences applications. This course is built on the pandas and xarray libraries and was developped through the Atelier Numérique of the Observatoire Midi-Pyrénées in Toulouse, France.

Supported by CNRS.

## Interactive Online Environment

You can run these notebooks online without installing anything using Binder:

1. Click the **launch binder** button above.
2. Wait for the environment to build (this may take a few minutes).
3. Once the **Welcome.ipynb** notebook opens, click the links to access the TPs and corrections.

> [!NOTE]
> The datasets are downloaded automatically when you run the notebooks. If you are hosting the large datasets (e.g., the 1GB GLORYS file), ensure the URLs in the notebooks are updated.
